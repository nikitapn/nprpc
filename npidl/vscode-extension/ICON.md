# Icon Placeholder

To add a custom icon for the extension:

1. Create a 128x128 PNG icon file
2. Save it as `icon.png` in this directory
3. Add this to `package.json`:
   ```json
   "icon": "icon.png"
   ```

For now, the extension will use the default VSCode icon.

Suggested icon design:
- A stylized "NP" monogram
- Colors: Blue/purple gradient (to match IDE themes)
- Simple, recognizable design at small sizes
